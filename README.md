# Netflix Data Analysis

This project performs exploratory data analysis on the Netflix dataset to uncover insights about content trends, genres, and release patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Analysis
- Movies vs TV Shows distribution
- Content release trends over the years
- Most popular genres on Netflix
- Country-wise content production
